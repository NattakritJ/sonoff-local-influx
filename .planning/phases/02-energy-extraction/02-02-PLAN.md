  <implementation>
    Add to `src/extractor.py` (do not remove Plan 01 code):

    1. Define `_MULTI_CHANNEL_UIIDS` dict:
       ```python
       _MULTI_CHANNEL_UIIDS = {
           126: 2,   # DualR3: 2 channels
           130: 4,   # SPM-4Relay: 4 channels
       }
       ```

    2. Define `_MULTI_PARAM_SUFFIXES`:
       ```python
       _MULTI_PARAM_SUFFIXES = ["_00", "_01", "_02", "_03"]
       ```

    3. Implement `extract_energy_multi(device_id: str, uiid: int, params: dict) -> list[EnergyReading]`:
       - If `uiid not in _MULTI_CHANNEL_UIIDS`: return `[]`
       - `num_channels = _MULTI_CHANNEL_UIIDS[uiid]`
       - `readings = []`
       - For `ch_idx` in `range(num_channels)`:
         - `suffix = _MULTI_PARAM_SUFFIXES[ch_idx]`
         - `raw_power   = params.get(f"actPow{suffix}")`
         - `raw_current = params.get(f"current{suffix}")`
         - `raw_voltage = params.get(f"voltage{suffix}")`
         - If all three are None → `continue` (skip absent channel)
         - Scale each ×0.01: `power = round(_to_float(raw_power) * 0.01, 4) if raw_power is not None else None`
         - Append `EnergyReading(device_id, uiid, power, voltage, current, energy_today=None, channel=ch_idx + 1)`
       - Return `readings`

    Note: Reuse `_to_float()` from Plan 01. No new dependencies needed.
  </implementation>
</feature>
