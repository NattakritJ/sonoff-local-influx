**Class: `SonoffDaemon`**

```python
class SonoffDaemon:
    def __init__(self, devices: list[DeviceConfig], writer: InfluxWriter) -> None:
        self._devices = {d["device_id"]: d for d in devices}
        self._writer = writer
        self._write_count = 0
        self._shutdown = asyncio.Event()
```

**`async def run(self) -> None`:**
1. Set up SIGTERM/SIGINT handlers pointing to `self._shutdown.set()`
2. Create `aiohttp.ClientSession` + `AsyncZeroconf` + `XRegistryLocal`
3. Call `registry.dispatcher_connect(SIGNAL_UPDATE, self._on_update)`
4. Start mDNS: `registry.start(azc.zeroconf)`
5. Log startup: `"SonoffLAN-InfluxDB ready | devices=%d | influx=%s | ids=%s"`
6. Launch heartbeat as a background `asyncio.Task`
7. `await self._shutdown.wait()`
8. Cancel heartbeat task, await it (suppress `CancelledError`)
9. `await registry.stop()` + `await azc.async_close()` + `self._writer.close()`
10. Log `"Daemon stopped cleanly."`

**`def _on_update(self, msg: dict) -> None`:**
- Filter: if `msg.get("deviceid")` not in `self._devices`, return
- Get `cfg = self._devices[deviceid]`
- Decrypt if `"data" in msg and "iv" in msg`, else use `msg.get("params", {})`
- Get `uiid = int(msg.get("uiid", 0))`
- Call `extract_energy_multi()` for multi-channel UIIDs (126, 130), else `extract_energy()`
- For each `EnergyReading` returned: schedule `asyncio.ensure_future(self._write_reading(r, cfg))`
- Log: `"UPDATE | %s (%s) | uiid=%d | params=%s"` at DEBUG if params non-empty
- If extract returns None/empty: no write, no log

**`async def _write_reading(self, reading: EnergyReading, cfg: DeviceConfig) -> None`:**
- `device_name = cfg.get("device_name", reading.device_id)`
- `await self._writer.write(reading, device_name=device_name)` 
- On success (no exception raised from write — it logs internally): increment `self._write_count`
- Log at INFO: `"WRITE | %s (%s) | ch=%s | power=%s W | voltage=%s V | current=%s A"` with `reading.channel or "-"`
- Note: writer.write() never raises — log the success line after calling write

**`async def _heartbeat(self) -> None`:**
```python
while True:
    await asyncio.sleep(60)
    _LOGGER.info("HEARTBEAT | writes=%d", self._write_count)
```

**`async def main() -> None`:**
```python
devices = parse_config()
host, token, database = parse_influx_config()
writer = InfluxWriter(host, token, database)
try:
    await writer.check_connectivity()
    _LOGGER.info("InfluxDB connected | host=%s | database=%s", host, database)
except RuntimeError as e:
    _LOGGER.error("Startup failed: %s", e)
    sys.exit(1)

daemon = SonoffDaemon(devices, writer)
await daemon.run()
```
