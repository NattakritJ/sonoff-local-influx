  <action>
    In `src/config.py`, add `parse_log_level()` below `parse_influx_config()`:

    ```python
    _VALID_LOG_LEVELS = {"DEBUG", "INFO", "WARNING", "ERROR", "CRITICAL"}

    def parse_log_level() -> int:
        """Read LOG_LEVEL env var and return the corresponding logging int.

        Valid values (case-insensitive): DEBUG, INFO, WARNING, ERROR, CRITICAL.
        Defaults to INFO if not set. Calls sys.exit(1) on invalid values.
        """
        raw = os.environ.get("LOG_LEVEL", "INFO").upper()
        if raw not in _VALID_LOG_LEVELS:
            print(
                f"ERROR: LOG_LEVEL={raw!r} is not valid.\n"
                f"  Allowed values: {', '.join(sorted(_VALID_LOG_LEVELS))}",
                file=sys.stderr,
            )
            sys.exit(1)
        return getattr(logging, raw)
    ```

    In `src/__main__.py`, update the `main()` function to call `parse_log_level()` before `logging.basicConfig`:

    - Import `parse_log_level` alongside existing config imports (add to the existing import line from config).
    - In `main()`, add before `logging.basicConfig(...)`:
      ```python
      log_level = parse_log_level()
      ```
    - Change `level=logging.INFO` in `logging.basicConfig(...)` to `level=log_level`.

    Note: `logging.basicConfig` is called at module level currently (line 16). Move it inside `main()` (called after `_load_dotenv()`) so `LOG_LEVEL` is read after the `.env` file is loaded. The call order in `main()` must be:
    1. `_load_dotenv()`
    2. `log_level = parse_log_level()`
    3. `logging.basicConfig(level=log_level, ...)`
    4. existing `parse_config()`, `parse_influx_config()`, etc.
