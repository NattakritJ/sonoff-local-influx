<objective>
Round power, voltage, and current fields to 2 decimal places in the energy extraction layer to eliminate floating-point precision artifacts (e.g. `1315.6100000000001` → `1315.61`).

Purpose: Numeric values written to InfluxDB should be clean 2dp floats, not IEEE 754 artifacts. Rounding belongs in the extraction layer, not the write layer, so EnergyReading is always clean.
Output: Updated `src/extractor.py` with `round(..., 2)` on power/current/voltage; new regression tests in `tests/test_extractor.py`.
</objective>
